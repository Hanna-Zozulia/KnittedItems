# Сайт каталог вязаных изделий

Простое веб-приложение для просмотра каталога вязаных изделий, созданное с использованием Node.js, Express и Sequelize.

## Технологии

- **Node.js**: Среда выполнения JavaScript.
- **Express**: Веб-фреймворк для Node.js.
- **EJS**: Шаблонизатор для генерации HTML.
- **Sequelize**: ORM для работы с базами данных.
- **MySQL**: Реляционная база данных.
- **Jest**: Фреймворк для тестирования.
- **Dotenv**: Для управления переменными окружения.

## Установка и запуск

### 1. Клонируйте репозиторий

```bash

git clone
```

### 2. Установите зависимости

Выполните команду для установки всех необходимых пакетов:

```bash
npm install
```

### 3. Настройте переменные окружения

Создайте файл `.env` в корневой папке проекта и добавьте в него следующие переменные для подключения к вашей базе данных MySQL:

```
DB_HOST=localhost
DB_USER=ваше_имя_пользователя
DB_PASSWORD=ваш_пароль
DB_NAME=имя_вашей_базы_данных
DB_DIALECT=mysql
```

### 4. Запустите приложение

После завершения установки и настройки, запустите сервер:

```bash
node app.js
```

Приложение будет доступно по адресу `http://localhost:3000`.

## Тестирование

В проекте настроены модульные тесты для контроллеров с использованием Jest. Для запуска тестов выполните команду:

```bash
npm test
```
